Справка по загрузчику

Добро пожаловать в @@@product@@@

Используйте это меню для выбора нужной функции. Если имеются проблемы с навигацией в справочной системе, нажмите @@@fkey_help@@@ для входа в описание справочной системы. Основные функции меню:

Запуск живой системы: Выберите этот вариант для запуска с использованием живого диска @@@product@@@.

Запуск (несвободные драйверы): Выберите этот вариант для запуска с использованием живого диска @@@product@@@, включая несвободные драйверы.

Загрузка с жесткого диска: Этот вариант не влияет на установленную систему. Он лишь запускает ранее установленную операционную систему.

Программа обнаружения устройств: Этот вариант запускает средство проверки аппаратного обеспечения. Это средство не оказывает влияние и не определяет аппаратное обеспечение, используемое при установке @@@product@@@.

ACPI отключено: Многие современные компьютеры имеют неполную или дефектную реализацию ACPI. Этот вариант отключает поддержку ACPI ядра, но включает многие свойства, такие как DMA для жестких дисков IDE.

Локальное APIC отключено: Если не удается осуществить установку обычным способом, это может возникнуть из-за системного аппаратного обеспечения, не поддерживающего локальное APIC. Если вы думаете, что причина в этом, используйте вариант для установки без поддержки локального APIC.

Проверка памяти: Проверка памяти полезна не только для испытания установки новых модулей памяти. Это стресс-тест для большей части компьютерной системы, позволяющий выявить проблемы с аппаратным обеспечением.

Параметры загрузки: Параметры загрузки могут полностью изменить поведение системы. Они являются настройками ядра.

@@@fkey_help@@@ Справка: Зависит от контекста. Она показывает различное содержимое в зависимости от активного элемента загрузочного экрана. Имеется также описание справочной системы.

@@@fkey_lang@@@: Установка языка и раскладки клавиатуры, используемых загрузчиком.

@@@fkey_video@@@ Разрешение: Здесь выберите между различными разрешениями экрана во время установки. Если имеются проблемы с графической установкой, их можно обойти использованием текстового режима.
