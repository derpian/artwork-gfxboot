Запуск живой системы @@@product@@@ с несвободными драйверами

Выберите этот вариант для запуска с использованием живого диска @@@product@@@. При использовании данного варианта @@@product@@@ попытается использовать несвободные драйверы для аппаратного обеспечения, такие как проприетарные драйверы nVidia или Ati Catalyst.
