Загрузка с жесткого диска

Выберите Загрузка установленной ОС для запуска системы, установленной на локальном жестком диске. Система должна быть установлена правильно, потому что MBR (Master Boot Record) запускается только с первого жесткого диска. ID устройства первого жесткого диска предоставляет BIOS компьютера. Если живая система запущена с USB, указанное hd0 должно быть изменено на hd1.

Используйте это, если забыли удалить CD или DVD из привода и хотите запустить компьютер с жесткого диска.
